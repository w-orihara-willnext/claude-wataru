# Claude Code ルール
## 作業範囲
- 作業は `~/Desktop/claude-wataru` フォルダ内のみに限定する

## アクセス禁止
- `/System`, `/usr`, `/bin`, `/sbin`, `~/Library` には絶対アクセスしない
- システムファイルや実行ファイル（`.exe`, `.app`, `.sh`, `.bash` 等）には触らない

## ファイル操作
- ファイルを削除する前に必ずユーザーに確認を取る
- ファイルを更新する際は、更新前の古いバージョンを必ず `old/` フォルダに移動してから上書きする

## ネットワーク
- 外部へのネットワークリクエストを行う前に、その内容をユーザーに報告する

## Git同期ルール
- 作業前に必ず git pull を実行する
- 作業後は以下を実行する：
  git add .
  git commit -m "作業内容のメモ"
  git push
